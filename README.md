# Umbra

Unofficial FFXIV launcher for 1.x clients.

**NOTE:** This application is still in the early stages of development, and the game may be unreliable to launch.

![Main Screenshot](https://xiv.zone/software/umbra/main-screenshot.png)

## Get It

Details on where to find stable releases of Umbra can be found on its [homepage](https://xiv.zone/umbra/install).

## Building

Please refer to the [building document](BUILDING.md) for instructions on how to build Umbra.

## Contributing & Support

Submitting patches to help fix bugs or add functionality is always appreciated. Filing issues is good too, but I do this in my free time so please don't expect professional support.
